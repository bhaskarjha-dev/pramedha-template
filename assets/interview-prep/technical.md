## Take-Home / Live Coding Checklist

Before submitting any take-home assignment:
- [ ] README with setup instructions
- [ ] Tests written and passing
- [ ] Edge cases handled
- [ ] Code is clean, commented where non-obvious
- [ ] Time complexity documented for key functions
- [ ] Git history shows incremental progress (not one giant commit)
